About This Project

The Placement Prediction System is a machine learning web application designed to assist students, faculty, and placement officers in identifying placement readiness based on various academic and skill-based parameters.

Key Objectives

  • Provide early insights into student placement possibilities.
  • Encourage students to improve key skill areas like soft skills, aptitude, and certifications.
  • Support decision-making for training and development efforts.

How It Works

The system uses trained machine learning models like Logistic Regression, Random Forest, and Support Vector Machines to analyze input data including CGPA, internships, certifications, soft skills, and more, then predicts whether a student is likely to be placed.

Technologies Used

  • Python & Flask (Backend)
  • Scikit-learn for ML Models
  • HTML, CSS for UI

This is a student project and intended for educational purposes.
